# YOUR_NEWS

YOUR_NEWS is an AI-powered personalized news recommendation system that delivers real-time news content based on user interests, preferences, and interactions.

The platform is designed to provide a smarter, more engaging, and user-friendly news experience with features like personalized recommendations, voice search and interactive news filtering.

---

# Features

- Personalized News Feed
- User Authentication (Login/Register)
- AI-Based News Recommendations
- Voice Search Functionality
- Interactive Like/Dislike System
- Responsive User Interface
- Real-Time News Updates
- Chatbot as News Assistant
- Customizable News Filtering

# Installation and Setup

## 1. Clone the Repository

```bash
git clone <your-github-repository-link>
cd YOUR_NEWS
```

---

# Backend Setup

## 2. Navigate to Backend Folder

```bash
cd backend
```

## 3. Install Backend Dependencies

```bash
npm install
```

## 4. Create .env File

Create a `.env` file inside the backend folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
NEWS_API_KEY=your_news_api_key
GEMINI_API_KEY=your_gemini_api_key
```

## 5. Run Backend Server

```bash
npm start
```

or (if using nodemon)

```bash
npm run dev
```

Backend will start on:

```bash
http://localhost:PORT
```

---

# Frontend Setup

## 6. Open a New Terminal and Navigate to Frontend Folder

```bash
cd frontend
```

## 7. Install Frontend Dependencies

```bash
npm install
```

## 8. Run React Frontend

```bash
npm start
```

Frontend will start on:

```bash
http://localhost:PORT
```
